# Pyton-course
В файле представлено решение следующей задачи: Напишите программу, которая на вход будет принимать список, элементы которого могут быть любыми. В данном массиве необходимо найти самую длинную строку, если таких будет несколько, то выбрать нужно первую. В найденной строке необходимо найти самую длинную подстроку без повторений. Таких строк может быть несколько, но возвращать функция должна первую подстроку. Результатом функции должно быть два объекта: индекс самой длинной строки списка и самая длинная подстрока без повторений. Если в исходном списке нет строк, то программа должна вернуть индекс -1 и пустую строку.

Программа должна состоять из трех функций: check_input, get_substring и main.

Функция check_input проверяет элемент массива и возвращает True, если элемент – строка, иначе – False.

Функция get_substring находит и возвращает самую длинную подстроку без повторений в переданной строке.

Функция main принимает на вход список, находит самую длинную строку списка, полученную строку передает на метод get_substring и печатает на экран (с помощью print) индекс самой длинной строки в массиве и найденную подстроку.


Были выполнены условия задания, созданы 3 функции. Функция check_input спользует встроенную проверку типов. Функция get_substring использует вложенные циклы для перебора всех возможных вариантов подстрок. Внутренний цикл собирает символы по порядку, пока не встретит повтор. Если новая найденная подстрока строго длиннее предыдущей, мы её сохраняем. Функция main - enumerate(data_list) позволяет одновременно получать и сам элемент, и его индекс. longest_str и target_index обновляются, только если находим строку, длина которой строго больше текущего максимума. Это гарантирует, что если в списке будут две строки одинаковой максимальной длины, программа выберет первую, как указано в условии.
Если после цикла target_index остался равен -1, значит, строк в списке не было.
